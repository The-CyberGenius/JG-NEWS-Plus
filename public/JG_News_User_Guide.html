🌐 Section 1: Website और Admin Panel क्या है?

आपकी JG News Plus Rajasthan website के दो हिस्से हैं:

हिस्सा क्या है? URL
🌐 Public Website जो आम लोग देखते हैं — खबरें, Live TV, Gallery localhost:5173
🔐 Admin Panel सिर्फ आप देख सकते हैं — खबरें add/edit/delete करना localhost:5173/admin
✅ याद रखें: Public website पर कोई Admin link नहीं है। Admin Panel सिर्फ URL से खुलता है।
🔐 Section 2: Admin Panel में Login
  1. 1
    Browser खोलें
    Chrome, Firefox, या Safari — कोई भी browser खोलें।
  2. 2
    Admin URL Type करें
    Address bar में यह URL type करें:
    http://localhost:5173/admin
  3. 3
    Password Enter करें
    Login page पर आपको यह credentials चाहिए:
    Username:admin
    Password:jgnews@2024
  4. 4
    "Login करें" बटन दबाएं

⚠️ ध्यान दें: Password गलत होने पर "गलत पासवर्ड" message आएगा। ऊपर दिया password ध्यान से type करें।
➕ Section 3: नई खबर कैसे जोड़ें

Login होने के बाद Dashboard में जाएं। फिर इन steps follow करें:

  1. 1
    Left Sidebar में "➕ खबर जोड़ें" पर Click करें
    या Dashboard में Quick Action button से भी जा सकते हैं।
  2. 2
    Form में सारी जानकारी भरें
    नीचे सभी fields की जानकारी दी गई है:
📝 शीर्षक (Title) *जरूरी
खबर का मुख्य शीर्षक हिंदी या अंग्रेजी में।
उदाहरण: "जयपुर में भारी बारिश"
📄 सारांश (Excerpt)
खबर का 2-3 लाइन का संक्षिप्त सारांश। होमपेज पर यही दिखेगा।
📰 पूर्ण खबर (Content) *जरूरी
पूरी खबर यहाँ लिखें। <p> टैग से paragraph बना सकते हैं।
🏷️ श्रेणी (Category) *जरूरी
Dropdown से चुनें: राजनीति, खेल, अपराध, मनोरंजन आदि।
📍 स्थान (Location) *जरूरी
राजस्थान के 33 जिलों में से चुनें। Dropdown में सभी हैं।
✍️ लेखक (Author)
खबर किसने लिखी — जैसे "JG News Desk" या रिपोर्टर का नाम।
🏷️ टैग (Tags)
खबर से जुड़े शब्द, comma से अलग करें।
उदाहरण: जयपुर, बारिश, मानसून
📷 फोटो (Image)
Image URL paste करें या "फाइल अपलोड" से अपने computer से photo चुनें।
🎬 Video URL
YouTube video embed link paste करें।
Format: youtube.com/embed/VIDEO_ID
  1. 3
    Breaking News और Featured Toggle Set करें
    🔴 Breaking News — ON करने से खबर ऊपर के red ticker में चलेगी।
    ⭐ Featured — ON करने से खबर होमपेज पर बड़ी दिखेगी।
  2. 4
    "✅ खबर प्रकाशित करें" बटन दबाएं
    खबर तुरंत website पर live हो जाएगी! 🎉
💡 Tip: खबर publish होने के बाद Homepage खोलकर देखें — आपकी नई खबर तुरंत दिखेगी।
✏️ Section 4: खबर Edit/Delete करना
  1. 1
    Sidebar में "📰 खबरें प्रबंधन" पर Click करें
    सभी खबरों की list दिखेगी।
  2. 2
    खबर खोजें
    Category filter से filter करें या ऊपर-नीचे scroll करके खबर ढूंढें।
  3. 3
    Edit या Delete करें
    ✏️ Edit — बटन से खबर खोलें और बदलाव करें, फिर "Update" दबाएं।

    🗑️ Delete — बटन से खबर हटाएं। Confirm popup आएगा — "हटाएं" दबाएं।
⚠️ सावधान: Delete की गई खबर वापस नहीं आती। Delete से पहले दो बार सोचें।
🔴 Section 5: Breaking News Set करना

Breaking News वो होती है जो website के ऊपर red ticker में चलती है।

  1. 1
    "📰 खबरें प्रबंधन" में जाएं
  2. 2
    खबर के सामने "Breaking" बटन देखें
    अगर 🔴 हाँ है — Breaking ON है।
    अगर "नहीं" है — Breaking OFF है।
  3. 3
    बटन Click करके Toggle करें
    तुरंत website के ticker में बदलाव दिखेगा।
💡 Best Practice: एक समय में 3-5 से ज़्यादा Breaking News न रखें। Ticker बहुत लंबा हो जाता है।
📺 Section 6: Live TV URL बदलना
  1. 1
    Sidebar में "📺 Live TV" पर Click करें
  2. 2
    YouTube Live URL Paste करें
    YouTube पर अपना Live video खोलें। URL कॉपी करें।
    जैसे: https://youtube.com/watch?v=ABC123XYZ
  3. 3
    "💾 Save करें" दबाएं
    System auto convert करेगा। Preview भी दिखेगा। ✅
ℹ️ जानकारी: Regular YouTube URL paste भी काम करता है — system खुद Embed format में convert कर देता है।
🏷️ Section 7: Categories Manage करना
  1. 1
    Sidebar में "🏷️ श्रेणियाँ" पर Click करें
  2. 2
    नई Category जोड़ने के लिए
    Input box में category का नाम हिंदी में लिखें → "➕ जोड़ें" दबाएं।
  3. 3
    Category हटाने के लिए
    Category के सामने "🗑️ हटाएं" → Confirm popup → "हटाएं" दबाएं।
🌐 Section 8: Hindi/English Language Switch

Website पर Visitors भाषा बदल सकते हैं:

  1. 1
    Website के Header (ऊपर) में देखें
    एक "🌐 EN" या "🌐 हि" बटन दिखेगा।
  2. 2
    Click करें
    Website की भाषा तुरंत बदल जाएगी — Hindi ↔ English। Browser remember रखेगा।
💡 Note: News articles का content वही दिखेगा जो आपने लिखा है। केवल Navigation, Buttons, और Labels बदलते हैं।
